# Source Registry

This document details the data sources utilized in the "NJ Interest Rates and Housing Affordability Study." All sources are sourced from the Federal Reserve Economic Data (FRED) API, ensuring consistency and reliability in macroeconomic data.

## Registered Sources:

### 1. New Jersey State House Price Index (NJSTHPI)
*   **Source Key:** NJSTHPI
*   **Type:** API
*   **Origin:** [FRED - NJSTHPI](https://fred.stlouisfed.org/series/NJSTHPI)
*   **Justification:** This series is crucial for tracking house price growth specifically within New Jersey, directly addressing the core objective of analyzing housing affordability in the state.

### 2. New Jersey Unemployment Rate (NJURN)
*   **Source Key:** NJURN
*   **Type:** API
*   **Origin:** [FRED - NJURN](https://fred.stlouisfed.org/series/NJURN)
*   **Justification:** The unemployment rate is a key economic indicator influencing housing demand and affordability. Including NJURN allows for the examination of its correlation with housing market dynamics in New Jersey.

### 3. Consumer Price Index for All Urban Consumers: All Items in U.S. City Average (CPIAUCSL)
*   **Source Key:** CPIAUCSL
*   **Type:** API
*   **Origin:** [FRED - CPIAUCSL](https://fred.stlouisfed.org/series/CPIAUCSL)
*   **Justification:** CPIAUCSL is essential for adjusting housing prices for inflation, enabling the computation of real (inflation-adjusted) affordability change over the study period. It provides the necessary benchmark for understanding the true purchasing power impact on housing.

## Summary:
A total of 3 sources have been registered, each directly supporting the research questions outlined in the project brief.
