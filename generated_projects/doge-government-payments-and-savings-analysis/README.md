# DOGE Government Payments and Savings Analysis

This project explores the live DOGE API as a research-grade source for federal
payments and reported efficiency savings.

Current focus:

- profile payment activity exposed by `/payments` and `/payments/statistics`
- document the coverage and limitations of the payment dataset
- compare the structure of payments with DOGE-reported savings feeds for grants,
  contracts, and leases
- prepare the workspace for later dashboards, audits, or NLP review of payment
  justifications

Primary API documentation:

- Swagger UI: https://api.doge.gov/docs
- OpenAPI spec: https://api.doge.gov/openapi.json
