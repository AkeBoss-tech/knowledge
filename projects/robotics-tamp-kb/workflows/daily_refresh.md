# Daily refresh workflow

1. Check recent research emails from Akash for new notes, links, or papers.
2. Check for 1-3 meaningful new planning-system / TAMP / dual-arm items.
3. Add raw material to `topics/inbox/` with date-stamped filenames.
4. Update or create one synthesized note in `topics/notes/`.
5. Commit and push if there is a meaningful change.
6. Keep the update concise and auditable; avoid dumping low-signal links.
