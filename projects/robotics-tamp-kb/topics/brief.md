# Research Brief

Akash wants a serious working knowledge base for robotics planning — especially dual-arm task-and-motion planning, scheduling, execution, and the practical bottlenecks that break elegant symbolic plans in real systems.

This project should favor:

- concrete systems over vague hype
- papers that explain failure modes, not just wins
- open-source packages that are usable in practice
- notes that translate research into buildable project directions
- a running memory of what is actually worth learning deeply
