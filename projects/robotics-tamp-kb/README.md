# Robotics TAMP Knowledge Base

A local-first KRAIL-style knowledge project for robotics planning with an initial focus on:

- dual-arm task and motion planning
- task-motion-scheduling
- PDDLStream and related solvers
- graphs of convex sets / IRIS / Drake planning geometry
- behavior trees and execution-layer control
- practical benchmarks for rearrangement, assembly, sorting, handoff, and long-horizon coordination

## Current status

This project was seeded from Akash's June 6, 2026 research email and lightly verified against a small set of primary sources.
