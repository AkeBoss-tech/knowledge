# Acme Metrics Brief

Acme Metrics is a fictional B2B analytics company used to demonstrate the
`company-brain` pack.

The goal of this project is to maintain a lightweight operating memory for:

- company overview and operating model
- teams, systems, and workflows
- recurring executive brief generation
- stale document and source review

The initial company brain should answer:

- what teams own which systems
- which workflows are governed by policy
- which notes should be refreshed after a source changes
- what a weekly executive brief should summarize
